Cacao y cafe

Tienda simulada de cafe colombiano hecha para cubrir los tres entregables arbol de componentes y gestion de estado con props.

React Vite JSX Props Mock data

1Proyecto React inicializado

Creado con npm create vite@latest, plantilla React, y organizado en carpetas según buenas practicas.

tienda-cacao/ ── src/ │ ├── assets/ recursos estaticos │ ├── components/ Navbar, Hero, ProductCard, Footer │ ├── data/ products.js (mock data) │ ├── views/ Home.jsx │ ├── App.jsx │ └── index.css ├── index.html └── package.json

2arbol de componentes

La interfaz se dividio logicamente, sin un solo archivo gigante

App ├─ Navbar (recibe cartCount por props) ├─ Home │ ├─ Hero │ └─ ProductCard N (uno por producto) Footer

Navbar

Marca, links de navegacion y contador de carrito.

Hero

Seccion principal de presentacion de la marca.

ProductCard

Muestra un producto individual: nombre, origen, notas y precio.

Footer

Enlaces, contacto y datos de cierre de pagina.

3Gestion de estado y props

La interfaz no esta vacia: el catalogo se arma a partir de un arreglo de datos simulados.

  • El mock data (6 productos) vive en src/data/products.js
  • Se importa en el componente padre Home.jsx
  • Se reparte a cada <ProductCard producto={'{producto}'} /> via props
  • Navbar recibe cartCount por props desde App
